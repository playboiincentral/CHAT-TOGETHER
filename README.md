Developed a social networking application that connects strangers through random matching, enabling them to chat, build friendships, and maintain long-term conversations.

### Key Features

* Implemented a real-time user matchmaking system using Firebase and Cloud Functions.
* Developed a one-to-one real-time chat system with message synchronization across devices.
* Enabled users to add friends and continue conversations after being matched.
* Integrated an AI Assistant within chat rooms to support user interactions.
* Built a push notification system using Firebase Cloud Messaging (FCM) to notify users of new messages.
* Implemented user moderation features, including blocking and reporting users.
* Developed an automated avatar scanning mechanism to detect inappropriate profile images.
* Built an admin moderation workflow for handling user reports and content violations.
* Added haptic feedback to enhance user interaction and application responsiveness.
* Designed and developed the application using SwiftUI and MVVM architecture.
